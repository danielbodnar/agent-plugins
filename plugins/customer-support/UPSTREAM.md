# customer-support

This plugin is **catalog-only**. Skill content is not vendored here because it is third-party.

To make it installable, either:

1. Copy the upstream skill directories into `./skills/` in this folder, or
2. Repoint the marketplace `source` for `customer-support` to the upstream repository that owns it.

Skills indexed in this bundle: customer-escalation, customer-research, draft-response, kb-article, ticket-triage.
