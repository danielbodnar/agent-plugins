# sp-global

This plugin is **catalog-only**. Skill content is not vendored here because it is third-party.

To make it installable, either:

1. Copy the upstream skill directories into `./skills/` in this folder, or
2. Repoint the marketplace `source` for `sp-global` to the upstream repository that owns it.

Skills indexed in this bundle: earnings-preview-single, funding-digest, sp-capital-iq-excel-pro, tear-sheet.
