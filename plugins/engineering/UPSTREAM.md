# engineering

This plugin is **catalog-only**. Skill content is not vendored here because it is third-party.

To make it installable, either:

1. Copy the upstream skill directories into `./skills/` in this folder, or
2. Repoint the marketplace `source` for `engineering` to the upstream repository that owns it.

Skills indexed in this bundle: architecture, code-review, debug, deploy-checklist, documentation, incident-response, standup, system-design, tech-debt, testing-strategy.
